# 🎬 Top 500 Movies Analysis | Power BI Dashboard

## 📖 Project Overview

This project is a Power BI dashboard that explores the Top 500 Movies ranked across popular platforms such as IMDb, Letterboxd, and Rotten Tomatoes. The dashboard focuses on comparing audience and critics ratings while uncovering trends related to genres, directors, languages, movie rankings, and production years.

As my first Power BI project, this dashboard challenged me to think critically about data storytelling, visualization design, and user experience. Despite the learning curve, I thoroughly enjoyed transforming raw data into meaningful insights.

---

## 🎯 Objectives

- Analyze the relationship between audience and critics ratings.
- Identify top-performing movies based on rankings.
- Explore movie distribution across genres and languages.
- Discover trends in movie production over time.
- Highlight directors with the highest number of productions in the dataset.

---

## 📊 Dashboard Features

### Key Metrics
- Average Audience Rating
- Average Critics Rating
- Average IMDb Votes

### Visualizations
- Audience vs Critics Ratings by Genre
- Top 5 Movies by Rank
- Production Count by Year
- Top Directors by Production Count
- Movie Language Distribution

### Interactive Elements
- Genre Filter
- Dynamic Visual Exploration

---

## 🌟 Dashboard Highlight

The **pink-themed "Top 5 Movies by Rank" visualization** serves as the centerpiece of the dashboard. It inspired the overall design and helped create a visually engaging storytelling experience while drawing attention to the highest-ranked films in the dataset.

## 📈 Key Insights

- Audience and critics ratings show strong alignment across many genres.
- Drama-related genres dominate the highest-ranked movies.
- English-language films represent the majority of the dataset.
- Several renowned directors appear frequently among the top-ranked films.
