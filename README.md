🎬 Top 500 Movies Power BI Dashboard


Overview

This Power BI dashboard analyzes the Top 500 Movies ranked across IMDb, Letterboxd, and Rotten Tomatoes.

The dashboard focuses on:

Audience Ratings vs Critics Ratings

Top 5 Movies by Rank

Production Trends by Year

Top Directors by Production Count

Movie Languages Distribution


Average IMDb Votes

Key Insights

Average Audience Rating: 85.75

Average Critic Rating: 85.58

English-language films dominate the dataset.

Drama-related genres appear most frequently among highly ranked movies.
